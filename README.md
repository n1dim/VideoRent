# VideoRent
Веб-приложение для управления видеопрокатом

---

# 1. Перечень и описание сценариев использования.
## 1.1 Роль: клиент
### Просмотр списка фильмов
- Пользователь открывает главную страницу.
- Система отображает список фильмов.
- Пользователь выбирает фильм.

### Просмотр карточки фильма
- Пользователь открывает страницу фильма.
- Система отображает:
   - название,
   - компанию,
   - режиссера,
   - год выхода,
   - тип носителя,
   - стоимость,
   - количество доступных экземпляров.
  
### Регистрация
- Пользователь вводит номер телефона и пароль.
- Система создает учетную запись.

### Авторизация
- Пользователь вводит номер телефона и пароль.
- Система выполняет вход.

### Работа с личным кабинетом
- Просмотр личных данных.
- Просмотр истории проката.
- Запрос продления аренды.
- Изменение личной информации.

## 1.2 Роль: Администратор
### Управление фильмами
- Добавление фильма
- Редактирование фильма
- Удаление фильма

### Выдача фильма
- Администратор выбирает фильм.
- Вводит номер телефона клиента.
- Указывает срок возврата.
- Система фиксирует выдачу.

### Возврат фильма
- Администратор выбирает фильм.
- Вводит номер телефона клиента.
- Фиксирует возврат.

### Управление пользователями
- Просмотр списка пользователей
- Блокировка / разблокировка
- Разрешение на продление срока аренды

---

# 2. Перечень и описание страниц
## Главная страница
Отображается:
- список фильмов

Действия:
- переход к карточке фильма
- авторизация
- регистрация
- переход в личный кабинет
- переход к панели администратора (для админа)
- добавить/удалить фильм (для админа)

## Страница фильма
Отображается:
- название
- компания
- режиссер
- год выхода
- тип носителя
- стоимость
- количество экземпляров

Действия:
- редактирование (для админа)
- выдача (для админа)
- прием (для админа)

## Личный кабинет
Отображается:
- ФИО
- телефон
- адрес
- история проката

Действия:
- изменение данных
- смена пароля
- запрос продления

## Страница регистрации
Отображается:
- поле для ввода телефона
- поле для ввода пароля
- поле для подтверждения пароля

Действия:
- регистрация

## Страница авторизации
Отображается:
- поле для ввода телефона
- поле для ввода пароля

Действия:
- авторизация

## Панель администратора
Отображается:
- количество пользователей
- количество фильмов
- количество выданных носителей
- список пользователей

Действия:
- перейти на страницу пользователя

## Страница пользователя
Отображается:
- ФИО
- телефон
- адрес
- история проката

Действия:
- продлить аренду
- заблокировать пользователя, указать причину

# 3. Схема навигации
```
Главная  
├── Карточка фильма  
├── Авторизация  
├── Регистрация
├── Панель администратора (для админа)
    └── Список пользователей 
        └── Страница пользователя  
└── Личный кабинет
```
